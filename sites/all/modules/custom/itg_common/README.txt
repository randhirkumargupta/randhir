############### HOW TO ADD SMILEY ##########

Below are steps to add smiley

1) ADD IMAGE TO SMILY FOLDER UNDER THEME
2) ADD DESCRIPTION OF IMAGES IN ARRAY CKEDITOR.config.smiley_descriptions OF 
    FILE itg_common.emoji.js
3) ADD IMAGE FILE NAME IN ARRAY CKEDITOR.config.smiley_images OF FILE 
   itg_common.emoji.js
